# Frontend Web - Sistema de Reconhecimento Facial HDT Energy

## Instalação

1. Instale as dependências: `npm install`
2. Crie um arquivo `.env.local` baseado no `.env.local.example`
3. Execute o servidor de desenvolvimento: `npm run dev`
4. Acesse `http://localhost:3000`

## Build para Produção

```bash
npm run build
npm start
```
